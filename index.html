Valhalla

What is it?

Valhalla is a Dedicated server for Valheim written in C++.

Background

Several smaller precursor projects eventually led to what Valhalla is today.

I can't remember exactly when, but development started around ~2020 when I was getting into networking for the first time. One of my first projects was CPPNetworking, a C++ networking experiment with very basic client/server interactions. It is since left abandoned, a relic of the past. It goes to show how little I was familiar with back then.

Roadmap

There are some annoying bugs in the project, likely as a result of my inability to run normal unit tests (I'm disorganized).

Why?

I fell in love with Valheim. I also like C++. The current Valheim Dedicated server also sucks. Why not create a Valheim server in C++?
